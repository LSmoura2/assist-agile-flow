# AgileAI Assistant
AI-powered assistant for Agile and DevOps teams - 

**Mestrado em Informática e Gestão · 2025/2026**

## What it does

AgileAI Assistant helps software teams with 6 AI-powered features:

- Generate User Stories with Gherkin acceptance criteria
- - Prioritize Backlog by value, risk and effort
- - Meeting Summary with decisions and action items
- - Sprint Planning support and retrospective questions
- - Release Readiness go/no-go recommendation
- Test Suggestions per user story
