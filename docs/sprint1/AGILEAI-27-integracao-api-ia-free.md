# AGILEAI-27: Integração com API de IA Gratuita

## User Story
Como utilizador, quero que o sistema integre com uma API de IA gratuita (ex: Hugging Face, Groq), para que as funcionalidades de geração de texto funcionem sem custos adicionais.

## Critérios de Aceitação
- [ ] Integração com pelo menos uma API gratuita (Hugging Face ou Groq)
- [ ] Autenticação por API key configurável via variável de ambiente
- [ ] Tratamento de erros e fallback em caso de indisponibilidade
- [ ] Tempo de resposta médio < 10 segundos
- [ ] Testes de integração implementados
