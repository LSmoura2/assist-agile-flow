# AGILEAI-16: Gerar Sumário de Reunião

## User Story
Como utilizador, quero que a IA gere automaticamente um sumário estruturado de reuniões de equipa, para poupar tempo e garantir rastreabilidade.

## Critérios de Aceitação
- [ ] Aceita transcrição ou notas brutas como input
- [ ] Gera sumário com secções: Participantes, Tópicos, Decisões, Próximos Passos
- [ ] Exporta para Markdown e PDF
- [ ] Tempo de processamento < 30 segundos
